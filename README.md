# Vibe KPI Demo
A beginner-friendly Applied Analytics project demonstrating ETL and secure SQL querying.

## Run Instructions

1. **Install dependencies:**
   `pip install -r requirements.txt`

2. **Load the data (ETL):**
   `python src/etl_load_sqlite.py`

3. **Run the KPI analysis:**
   `python src/kpi_city.py`

4. **Run tests:**
   `pytest tests/test_kpi_city.py`
